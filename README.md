# AIVOA Website Manual Testing Project

## Project Overview
This project contains a complete manual testing report for the AIVOA website:
https://aivoa.ai/

The testing focused on identifying:
- Functional bugs
- UI/UX issues
- Form validation problems
- Navigation inconsistencies
- Broken links
- Responsiveness issues
- General usability improvements

---

## Testing Scope
The following areas were tested:

- Homepage
- Navigation menus
- Contact forms
- Buttons and links
- Responsive design
- UI consistency
- Validation messages
- User experience flow
- Accessibility observations

The repository includes:

- Test Cases Document
- Bug Reports

- Observations & Improvements
- Manual Testing Evidence
